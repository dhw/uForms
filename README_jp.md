# uForms
Windows FormsのようにUnityのエディタ拡張ウィンドウのデザインを作成するためのエディタ拡張。

## 主な機能
* GUIベースでのウィンドウデザイン作成
* uForms用の各コントロールクラスを利用したコードの書き出し
* uFormsに依存せずに独立して動作するベタなコードの書き出し
* XML形式でのデザインの入出力
